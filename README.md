# VoltMart – Retail Billing & Inventory Management System

VoltMart is a full-stack web application designed to help retail stores manage products, generate bills, and track sales efficiently.  
The system includes secure user authentication with email OTP verification and role-based access for administrators and cashiers.

---

## Features
- User registration and login
- Email OTP verification during registration
- Role-based access (Admin / Cashier)
- Product management (add, update, delete products)
- Billing system with cart functionality
- Order tracking
- Sales dashboard and reports

---

## Tech Stack

**Frontend**
- React.js
- Axios
- React Router
- CSS

**Backend**
- Spring Boot
- Hibernate / JPA
- REST APIs

**Database**
- MySQL

**Deployment**
- Frontend: Vercel
- Backend: Render
